# Отчеты по тестам роботов

В этой папке лежат опубликованные итоговые отчеты по ad-hoc прогонам роботов OsEngine.

Правила:
- финальный отчет всегда публикуется сюда;
- имя файла должно содержать дату запуска, имя робота и тикер;
- для каждого прогона ожидаются два файла:
  - `*.json` с полной структурой данных;
  - `*.md` с краткой человекочитаемой сводкой.

Пример:
- `2026-03-12_18-51-43__BreakEma__Сбербанк.json`
- `2026-03-12_18-51-43__BreakEma__Сбербанк.md`
